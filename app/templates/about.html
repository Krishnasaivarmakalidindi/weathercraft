About WeatherCraft

WeatherCraft is a modern, responsive weather app built with Flask, Tailwind CSS, and Alpine.js. It features smooth animations, dark mode, and an accessible, mobile‑first UI.

  • Fast, compact JSON API for SPA‑like updates
  • File cache with optional Redis to reduce API calls
  • Skeleton loaders and Lottie animations for delightful feedback
  • 3‑day forecast scroller and expandable details
  • Dark mode toggle with persisted preference
